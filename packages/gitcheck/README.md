# gitcheck-cli

分析目录下所有 github 项目的提交情况

## 使用

```

gitcheck -d .     查找当前目录
gitcheck -d ../     查找上级目录
gitcheck -d D:\code\frontend  查找D:\code\frontend目录

```

## option

```
Options:
  -h, --help       Display this message
  -v, --version    Display version number
  -d ,--dir <dir>  Parse folder (default: .)

```
